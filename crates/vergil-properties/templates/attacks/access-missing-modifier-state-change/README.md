# access-missing-modifier-state-change

Detects the most loss-dominant attack class in smart-contract history: a privileged state-changing function with no access modifier, letting any caller mutate critical state. The canonical real-world instance is the **Parity Multisig** freeze (Nov 2017, ~$280M, Ethereum) — `initWallet` was callable by anyone, enabling ownership seizure and a subsequent `kill()`. Vergil's negation property requires that every state-changing call be made by an authorized principal; the Halmos encoding spins up a helper Attacker contract at a non-owner address and asserts that any non-reverting mutation attempt leaves the protected slot unchanged. See `manifest.yaml` for full references (SWC-105, SWC-100, OWASP SC01:2026) and the catalog entry `notes/attack-patterns.md` §1.1.
