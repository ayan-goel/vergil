# input-missing-parameter-validation

Detects state-changing functions whose external inputs or caller identity are unvalidated. Joint with `logic-approval-not-revoked-after-cancel` — both flaws were present in the **Hedgey Finance** (Apr 19, 2024 — ~$44.7M combined Arbitrum + Ethereum) production bug, so the fixture surface is shared and each manifest's Halmos check exercises a different invariant. Vergil's negation property here requires that the cancel path reject calls from non-owners; the Halmos encoding has a non-owner Attacker contract attempt to cancel a plan the legitimate owner created. In the vulnerable fixture the attacker's call flips `plans[id].active` to false; in the clean fixture (`require(msg.sender == owner)`) the call reverts. See `manifest.yaml` for references (OWASP SC04:2026) and `notes/attack-patterns.md` §8.1.
